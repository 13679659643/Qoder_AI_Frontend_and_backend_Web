# 0. 验证原则

- **数据驱动**：所有验证必须基于实际数据结果，禁止"已验证"等无证据声明
- **对比验证**：核心度量值必须与已知正确值（Excel/SQL 直查）进行交叉对比
- **边界测试**：必须测试空值、零值、异常值等边界条件
- **展示证据**：每个验证步骤必须展示实际结果截图或数据
